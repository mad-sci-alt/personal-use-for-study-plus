### 高级通信：关注数据交换
- **按同步和阻塞与否划分**

|     | 方式一    | 方式二     |     |
| --- | ------ | ------- | --- |
|     | **同步** | **异步**  |     |
|     | **阻塞** | **非阻塞** |     |
一般来说异步一定是非阻塞的，因为异步阻塞没有意义
而同步既可以是阻塞也可以是非阻塞
