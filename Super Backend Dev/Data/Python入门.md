#### 前言
1.Java属于半解释半编译型语言，原因在于其JIT（Just In Time）机制，JVM会将频繁代码段直接编译成字节码缓存起来，使其运行起来速度接近C++。

Java程序运行逻辑：
- 编译阶段Java源码会被javac程序首先编译为.class文件交给JVM看
- 解释阶段 JVM将字节码翻译为机器码

#### python的优势
2.开发简单方便，在AI、数据分析等计算密集型场景中的能用到的库非常多（相较于Java等其他语言），使用上手时间成本开销少，且动态类型特性使得使用很方便。其作为胶水语言只是用来进行解释的，在AI领域还是用其作为工具来调用底层的C/C++代码进行高效运作
