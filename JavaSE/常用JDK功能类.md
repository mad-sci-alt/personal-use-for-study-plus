与时间有关：

| 方法名               | 返回类型            | 形参  | 功能                     | 来自顶级父类名称 |
| ----------------- | --------------- | --- | ---------------------- | -------- |
| currentTimeMillis | long            | 无   | 返回程序运行到此方法时以毫秒为单位的当前时间 | System   |
| getClass          | Class<?>（即运行时类） | 无   | 返回该实例的运行时类             | Object   |
