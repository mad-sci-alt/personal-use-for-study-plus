### Java的[[枚举类]]可以用来干什么
- 可以写入一组相关联的常量，比如一年的四个季节、一年的十二个月份、一个星期的七天、色谱中的颜色种类
- 写入格式为字符串，每个变量之间用逗号(英文)隔开
- enum枚举可以写在类中
- enum枚举中的每个枚举元素都同时具有public [[final]] [[static]]属性（作用是什么就不用我说了吧，老生常谈）
- 用枚举创建对象可以直接对象名➕构造器参数，但是枚举创建必须要写在所有成员方法、属性最前面

Enum类的方法

| 方法名       | 返回类型     | 形参     | 功能                               |
| --------- | -------- | ------ | -------------------------------- |
| toString  | String   |        | 默认获取当前对象名，可被重写                   |
| name      | String   |        | 获取当前对象名，且方法为[[final]]            |
| ordinal   | int      |        | 获取该对象在枚举中的定义的顺序，为数组位置（从0开始计数）    |
| values    | String[] |        | 该方法默认被隐藏了，返回当前枚举类的所有常量(枚举对象名)    |
| valueOf   |          | String | 在枚举中寻找与形参匹配的枚举对象名，找到后返回该对象，否则抛异常 |
| compareTo | int      | E o    | 返回调用对象名与形参对象名位置码之差               |
